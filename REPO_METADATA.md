## Security and Trust (for repo homepage or wiki)

Claude Command Center runs entirely on your local machine. It binds to 127.0.0.1, stores all data in ~/.claude-command-center/ as plain JSON, and makes no outbound network requests during normal operation. There is no telemetry, analytics, or phone-home behavior. Secrets found in scanned configs are redacted before storage. All user inputs are validated with Zod schemas. CI builds include SHA-256 checksums for release verification. See SECURITY.md for the vulnerability reporting policy and docs/security-threat-model.md for the full threat model.
